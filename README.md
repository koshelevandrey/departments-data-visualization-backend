# Departments Data Visualization

Серверная часть приложения для анализа и визуализации данных департаментов.
Тестовое задание для Sapiens solutions.

## Руководство по использованию

1. Установить зависимости
### `npm install`
2. Запустить приложение
### `npm run start`
Сервер будет доступен по адресу [http://localhost:8000](http://localhost:8000).

## Технический стек

- [TypeScript](https://www.typescriptlang.org/)
- [json-server](https://github.com/typicode/json-server) - библиотека для создания сервера на основе json-файла
